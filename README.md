# 💻 Operating Systems (SECR2043)

Welcome to my repository for the **SECR2043 Operating Systems** course at Universiti Teknologi Malaysia (UTM). This space serves as a comprehensive portfolio documenting both my hands-on laboratory experiments and our collective milestone achievements for the core group research project. 

---

## 📖 Course Overview
This course explores the fundamental principles governing modern operating systems. Through practical Linux command-line tasks and C-programming simulations, I studied core system architecture mechanisms including process lifecycle tracking, multi-threaded task management, file systems, security permissions, and algorithmic virtual memory translation schemes.

---

## 📂 Repository Structure & Project Breakdown

### 🔹 Project: Enterprise Network Design (Group Team: Y-LINK)
* **General Summary:** Working together as an infrastructure consulting team, we designed a secure, scalable, and budget-optimized network infrastructure for the proposed two-story Faculty of Computing building (Block N28B). The project involved designing physical floor plan layouts, conducting expert feasibility interviews, evaluating multi-vendor hardware solutions, mapping Cat 6 star topology cabling paths, and developing a structured Variable Length Subnet Masking (VLSM) IP allocation scheme to handle future institutional growth. All progressive milestone tasks (Tasks 1–5) are fully integrated into our master blueprint and presentation.
* **Core Deliverables:** * Consolidated Network Engineering Report (`/task 6.pdf`)
  * Team Contribution Assessments & Video Presentation (`/TASK 6B SECR1213.pdf`)

### 🔹 Lab Work: Packet Analysis & Packet Tracer Simulations
* **General Summary:** A collection of hands-on laboratory assignments focused on analyzing network layer protocols and configuring live data communications. The practical assessments involved tracking real-world application layer traffic (HTTP/DNS query and response message states) using the Wireshark packet sniffer tool, simulating transport layer TCP/UDP session multiplexing behaviors, configuring dynamic routing protocols (RIP) to manage isolated LAN subnets across multiple routers, and investigating switching operations like Address Resolution Protocol (ARP) table lookups within Cisco Packet Tracer environments.
* **Core Deliverables:** * Wireshark Application Layer Traffic Analysis (`/lab1_network_farranurzahin.pdf`)
  * Cisco Packet Tracer Transport, Network & Data Link Simulations (`/LAB 2 NETWORK FARRA.pdf`, `/LAB 3 NETWORK.pdf`, `/lab4_network_farranurzahin.pdf`)

## 💡 Course Reflection & Key Takeaways
This course bridged the gap between abstract programming scripts and physical machine hardware execution. Building the paging simulator alongside my group members highlighted the importance of clean memory boundaries and data mapping strategies, turning complex theoretical address lookups into real, working execution blocks. Simultaneously, navigating through individual terminal lab assessments taught me how to confidently interact with core system subsystems—giving me total control over process lifecycles, user privileges, and script automations. Ultimately, this combination of team project design and rigorous lab work has equipped me with the technical foundation and system intuition necessary to analyze, troubleshoot, and write optimal code close to the metal.
